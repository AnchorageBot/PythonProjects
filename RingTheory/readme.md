A ring is a commutative group under addition that has a second operation: multiplication

Example
* Z = integers
- - - -

Group operations
* Addition
* Subtraction (additive inverse)

Ring operations
* Addition
* Subtraction
* Multiplication
   
Field operations
* Addition
* Subtraction
* Mutltiplication
* Division (multiplicitive inverse)
